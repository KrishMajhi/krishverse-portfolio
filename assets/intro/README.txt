PUT YOUR INTRO ASSETS IN THIS FOLDER.

1. dialogue-box.png  (or .gif)
   → Your comic narration/dialogue box image.
   → Referenced in index.html as: assets/intro/dialogue-box.png

2. swing-character.gif
   → Your swinging character animation.
   → Referenced in index.html as: assets/intro/swing-character.gif

File names must match exactly (case-sensitive). If you want to use
different names or a .gif instead of .png for the dialogue box, just
update the matching src="..." attribute inside index.html (search for
"ciBoxImg" and "ciSwinger").

You can delete this file once your real assets are in place.
